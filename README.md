# Skills

A collection of reusable agent skills for developer workflows.

## Installation

Install the skill using the `skills` package manager:

```bash
npx skills@latest add LRNZ09/skills/[SKILL_NAME]
```

## Usage

After installation, invoke the skill through your agent. The exact command depends on your local environment.

## Available Skills

### Writing & Documentation

- `generate-pull-request-description` – Generates a reviewer-ready pull request description and test plan based on commits and the full diff from the branch's base.

    ```bash
    npx skills@latest add LRNZ09/skills/generate-pull-request-description
    ```
